# 🗣️ English ↔ Swahili Translator

## 📖 Overview
This project is a **bi‑directional translator** between English and Swahili.  
It leverages modern sequence‑to‑sequence models (CTranslate2 / Hugging Face) to provide accurate translations in both directions.  
The translator is designed to handle everyday sentences, with ongoing improvements for idioms, proverbs, and cultural nuance.

---

## 🚀 Features
- **Bi‑directional translation** → Translate English → Swahili and Swahili → English.   

---

## 🧩 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/swahili-english-translator.git
   cd swahili-english-translator
   ```
2. Install dependancies
```bash
pip install -r requirements.txt
```
